pyUnnenberg
===========

Dies ist die Bake für den Unnenberg-Turm (DB0LTG).

Vorgesehen:
  * Dauerton 1200 Hz
  * CW - Callsign + Lokator
  * JT65B zu ungeraden Minute
  * alle 8 Minuten ein SSTV-Bild
    * aufgenommen von der Kamera auf dem Turm
    * nur tagsüber
    * ggf. MartinM1 und PasokonP3
  * GPS-synchronisierte Uhr
