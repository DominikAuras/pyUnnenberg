pyUnnenberg
===========

Dies ist die Bake für den Unnenberg-Turm (DB0LTG).

Vorgesehen:
  * Dauerton 800 Hz
  * PI4 - Kodiert Callsign
  * CW - Callsign + Lokator
  * ungefähr alle 10 Minuten ein SSTV-Bild
    * aufgenommen von der Kamera auf dem Turm
    * nur tagsüber
    * MartinM1 und PasokonP3
  * GPS-synchronisierte Uhr
